# 🚀 Getting Started

## ✔️ Prerequisites

- Python 3.13+
- Git
- VS Code or any IDE

---

## ✔️ Clone Repository

```bash
git clone https://github.com/KRAJNISH-REBEL/mini-google-drive.git

cd mini-google-drive
```

---

## ✔️ Create Virtual Environment

```bash
python -m venv venv
```

---

## ✔️ Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

---

## ✔️ Install Dependencies

```bash
pip install django
```

---

## ✔️ Apply Database Migrations

```bash
python manage.py migrate
```

---

## ✔️ Run Development Server

```bash
python manage.py runserver
```

---

## ✔️ Open in Browser

```text
http://127.0.0.1:8000/
```
